## Create a project

Creates a new project. The `project_id` returned in the response is required to create tasks.

`POST /projects`

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `name` | string | Yes | The name of the project. Must be a non-empty string. |
| `description` | string | No | An optional description of the project's purpose. |

### Responses

| Status | Meaning |
|--------|---------|
| `201 Created` | Project created successfully. Returns the project object. |
| `400 Bad Request` | The request body is missing the `name` field or contains an invalid value. Check the `message` field in the response for details. |
| `401 Unauthorized` | The `Authorization` header is missing or the API key is invalid. |

### Example request

```bash
curl -X POST https://test-writechoice.onrender.com/projects \
  -H "Authorization: Bearer tf-live-4a8f2c1e9b3d7e6a" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "TaskFlow Documentation",
    "description": "Writing the official TaskFlow API docs."
  }'
```

### Example response — 201 Created

```json
{
  "project_id": "e06ad061-e5c7-4949-9ab7-788f68eab337",
  "name": "TaskFlow Documentation",
  "description": "Writing the official TaskFlow API docs.",
  "created_at": "2026-07-03T02:45:56.075Z"
}
```

### Error responses

**400 Bad Request**

```json
{
  "error": "Bad Request",
  "message": "The 'name' field is required and must be a non-empty string."
}
```

**401 Unauthorized**

```json
{
  "error": "Unauthorized",
  "message": "Missing or invalid API key. Provide a valid key in the Authorization header as: Bearer <api_key>"
}
```

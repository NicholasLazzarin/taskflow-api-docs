# TaskFlow API

TaskFlow is a task management API that allows you to create projects, organize tasks within them, track progress, and keep your team aligned.

## What you can do

- **Create projects** to group related work
- **Add tasks** to projects and assign titles and descriptions
- **Track progress** by updating task status as work moves forward
- **Clean up** by deleting tasks that are no longer needed

## How it works

TaskFlow is organized around two resources: **projects** and **tasks**.

A project is a container for tasks. Before you can create a task, you need a project to put it in. This dependency is intentional — every task in TaskFlow belongs to exactly one project.
